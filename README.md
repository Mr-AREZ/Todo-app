# 📝 Todo List - Drag & Drop Practice

A simple Todo List application built for learning and practicing **JavaScript Drag & Drop** functionality.

> **Note:** This project is for educational purposes only and serves as a playground to experiment with Drag & Drop interactions in JavaScript.

---

## 🚀 Features

- ✅ Add new tasks
- ✏️ Edit tasks
- 🗑️ Delete tasks
- 📦 Drag & Drop to reorder tasks
- 💾 Persistent data using Local Storage *(if implemented)*
- 📱 Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Mr-AREZ/Todo-app.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
